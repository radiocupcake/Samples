Britney's Website Building Cheat Sheet

A step-by-step guide to build a basic webpage using HTML and CSS.

Step 1

The first thing you should always do is get your folders and files created and organized. To do this, create a folder in a designated spot of your choosing. Pick an easy-to-remember folder name that will make the project easily identifiable by you. Somewhere within that folder, you'll need to have one HTML document to write your HTML code in, and one CSS document to style your HTML using CSS code. You can create these in your text editor or IDE by opening the folder and adding a file. The HTML file name must end with .html and likewise the CSS file must end with .css. Once you have your folders organized to your preferences, and you've created an HTML and CSS file, you can move on to the next step.

Step 2

Now you need to open your HTML file and start adding some code. You'll need to put your HTML declaration first. Then the opening and closing declaration tags. Within that you should have your opening and closing head tags, followed by opening and closing body tags. The head tags should contain your metadata, your title, and this is also where you'll link your CSS file. You can find most of this code on the Codecademy website and copy and paste it. Then move on to step 3.

Step 3

Now that your header is complete, you can start working on the body. This will be made up of a variety of tags, some of which I have outlined in the table below. These tags will contain all of your websites content such as text, images, and tables. Once you've done that, begin step 4.

Tag Content Format Required Attributes
h1-h6 headings text between opening and closing tags none
p text/paragraphs text between opening and closing tags none
br line break a single self-closing tag none
div division of sections opening and closing tags with various content in between none
img an image a single self-closing tag src & alt
ul a list with bullet points opening and closing tags with list items contained none
ol a numbered list opening and closing tags with list items contained none
li list items text between opening and closing tags none
table a table opening and closing tags with table rows, table headings & table data contained none
tr table rows opening and closing tags with table headings & table data contained none
th table headings text between opening and closing tags none
td table data text between opening and closing tags none

Step 4

With your HTML code written, you can now star styling your webpage. This is the fun part. Be creative. Add colors, borders, align stuff, try some different fonts, and change the sizes of your elements. Most importantly, have fun!!!
